# poor-man-twitter

## Technical stacks

- Front end
  - vue 3.2.41
  - moment 2.29.4
- Back end
  - django 4.1.3
  - django rest framework 3.14.0
  - django corsheaders 3.13.0
  - SQLite3

## Installation and run

```code
## run front end 
move to client directory
...
npm install
npm run dev

# localhost:5173

## run back end

move to root directory
...

python manage.py runserver

# localhost:8000/api/
```
